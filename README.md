# 🤖 MyDailyWork_Task1_RuleBasedChatbot  

A **simple rule-based chatbot** built using **Python** that responds to user queries such as greetings, time, date, Python basics, and more.  

---

## 📌 Project Description  
This project demonstrates a **rule-based chatbot** that interacts with users based on predefined rules and keywords.  
It showcases basic **Natural Language Processing (NLP)** concepts and simple conversation flow.  

---

## ✨ Features  
- 👋 Greets the user (`hi`, `hello`, `hey`)  
- 🧑 Introduces itself when asked  
- ❓ Answers basic queries  
- ⏰ Tells the current time (12-hour format)  
- 📅 Displays the current date and day  
- 🐍 Explains Python and its basics  
- 🙏 Responds politely when the user exits  

---

## 🛠️ Technologies Used  
- **Python**  
- **Datetime module**  

---

## ▶️ How to Run the Project  

1. **Clone the repository**  
   ```bash
   git clone <your-repo-link>
   ```

2. **Navigate to the project folder**  
   ```bash
   cd MyDailyWork_Task1_RuleBasedChatbot
   ```

3. **Run the chatbot**  
   ```bash
   python chatbot.py
   ```
